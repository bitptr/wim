wim

Installation
------------

::

    % python setup.py install

Development
-----------

Prerequisites (Debian)::

    % sudo apt-get install python-gi libwnck-3-* gir1.2-wnck-3.0

Prerequisites (OpenBSD)::

    % sudo pkg_add py-gobject libwnck

Virtualenv::

    % mkvirtualenv --system-site-packages wim
